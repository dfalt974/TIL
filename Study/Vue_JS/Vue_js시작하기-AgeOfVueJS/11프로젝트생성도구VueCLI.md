Vue CLI

https://cli.vuejs.org

CLI <- Command Line Interface. 명령어를 통한 특정 액션을 수행하는 도구.

Get Started -> Installation -> 

VSCode 터미널 - 새 터미널 - `node -v`

노드 버전 10 이상 확인하기

`npm -v`

6 버전 이상인거 확인하기

`npm install -g @vue/cli`



설치하는 라이브러리가 위치하는 폴더에 파일쓰기 권한 없을 때(관리자 권한 아니라서) 에러나면

`sudo npm install -g @vue/cli`
