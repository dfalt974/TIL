뷰의 템플릿 문법

뷰로 화면을 조작하는 방법. 

* 데이터 바인딩
* 디렉티브



#### 데이터 바인딩

뷰 인스턴스에서 정의한 속성들을 화면에 표시하는 방법

콧수염 괄호(Mustache Tag)

```html
<div>
    {{ message }}
</div>
```

```javascript
new Vue({
    data: {
        message: 'Hello Vue.js'
    }
})
```



#### 디렉티브

뷰로 화면의 요소를 더 쉽게 조작하기 위한 문법 

HTML 태그에서 일반적인 id, class 등 표준 속성을 제외하고 v- 붙는 속성들 => 뷰 디렉티브

```html
<div>
    Hello <span v-if="show">Vue.js</span>
</div>
```

```javascript
new Vue({
    data: {
        show: false
    }
})
```
